Tentamen

Nedan följer de uppgifter som behöver göras. Varje uppgift ska lösas inuti .problem-area diven. CSS:en ska skrivas i style.css och JavaScripten ska skrivas i script.js filen. Du får inte ändra någon CSS som är skriven i "do_not_touch.css" filen. Du får heller inte ändra någon HTML i denna filen som inte ligger under en .problem-area div. Bootstraps CSS är redan inladdat på sidan som hjälp för att lösa Bootstrap uppgifterna.
